  2. 提高效率的途径

    • 3.1 工具使用
      IDE工具选择：
      首推sublime text，其次配合php推荐使用的phpstorm

      辅助插件：
      emmet、js/css format、tinypng cli、uncss

    • 3.2 code snippets
      搜集和收藏常用的css、js代码段，不需反复书写同样的功能
      如：js模板替换、ajax请求封装、js数据格式化、css3动画和按钮

    • 3.3 善于搜索
      技术问题首选google搜索（shadowsocks、vpn）
      前端技术问题可以查API文档和去经常更新的前端技术博客

    • 3.4 切图工具
      同类型图片可以批量导出图片（PS脚本）
      重复操作可以使用批量动作（PS动作）

    • 3.5 文档书写总结
      markdown、hexo、云笔记

    • 3.6 使用新技术
